## 📜 Rules
### 운영 방식
- **진행 방식**
  - 매주 월요일 저녁
  - Discord에서 온라인 모임
- **문제 선정**
  - 1인당 1문제씩 선정
  - 총 3문제가 해당 주의 **필수 문제**로 설정
- **리뷰 방식**
  - 각자 PR 후 코드 리뷰
- **모임 당일**
  - 당일에 각자 **랜덤으로 한 문제 설명** 
  - 문제 접근 방식, 구현 포인트, 고민한 점 등을 자유롭게 공유
 
    
### 참여 규칙
- 매주 선정된 **모든 필수 문제를 마감 전까지 풀이하여 PR 제출**
- 풀이 코드에는 **주석** 또는 **README**로 핵심 설명 필수
- 매주 **수요일 중간 점검**을 통해 풀이 진행 상황 확인
